Projects

SketchUp Cafe

One of my school's ICT project was to make a model of a cafe in a web application called SketchUp. Although assets, like the vehicles and dishes, are taken from other's creations, the Cafe building is still my own work.

Blender Animation

Using the Blender application, I created a short, 10-second ad-like video for an imaginary wine product. This video consists of 250 frames, running at 25 frames per second.

Text-Based Python Game

One of my earlier projects when I started to code using Python 3 was this fighting game inspired by RPG-like battle sequences. With two gamemodes, you can choose to fight the computer or fight a friend on the same device!

Construct 3 Game

As an extracurricular project, I was instructed to create a game using the Construct 3 game engine. This is a survival-shooter game, where you have to collect as many coins as possible while avoiding or shooting any enemies in sight.
